# Galaxy Aliances API

1. Clonar proyecto
2. npm i
3. clonar el archivo .env.template y renombrarlo a .env
4. cambiar las variables de entorno

Levantar la base de datos
```
docker-compose up -d
```

6. npm run start:dev
